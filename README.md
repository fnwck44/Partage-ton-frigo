## Pour commencer
## Installation

Installation des librairies nécessaire:

```
$ pip install -r requirements.txt
```


## Lancer flask :
```
$ python Partage-ton-frigo/frigo.py
```

Votre site internet:  `http://127.0.0.1:5000`

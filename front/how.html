COMMENT UTILISER LE FRIGO ?

À PARTAGER
  • Tous les végétaux : fruits, légumes, ...
  • Tous les produits secs : biscuits, épicerie, ...
  • Tous les produits avec DLUO*, même raisonnablement dépassée
  • Pour les produits à DLC*, les produits encore emballés, non entamés

À ÉVITER
  • Les viandes, les poissons
  • Les plats cuisinés maisons
  • Les œufs de ferme (souvent non datés)
  • Le lait, les bouteilles ouvertes

QUAND PARTAGER ?
Quelques exemples :
Avant les départs (week end, vacances, ...)
Après un événement (buffet traiteur, soirées, ...)
